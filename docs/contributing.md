# Contributing to ConvoSynth

We welcome contributions to ConvoSynth! Here’s how you can get involved:

## How to Contribute

1. **Fork the Repository**: Start by forking the repository on GitHub.

2. **Clone the Repository**: Clone your forked repository to your local machine.

    ```bash
    git clone https://github.com/AnnudeepSatyaSai/ConvoSynth.git
    cd ConvoSynth
    ```

3. **Create a New Branch**: Create a new branch for your feature or bug fix.

    ```bash
    git checkout -b feature-branch-name
    ```

4. **Make Your Changes**: Implement your feature or fix.

5. **Write Tests**: Ensure that your changes are covered by appropriate tests.

6. **Commit and Push**: Commit your changes and push them to your fork.

    ```bash
    git add .
    git commit -m "Description of your changes"
    git push origin feature-branch-name
    ```

7. **Submit a Pull Request**: Open a pull request on the main repository.

## Guidelines

- **Code Quality**: Ensure your code adheres to the existing code style and is well-documented.
- **Testing**: Please add or update tests as appropriate.
- **Discussion**: If you plan to make significant changes, please open an issue first to discuss them.

Thank you for contributing to ConvoSynth!
